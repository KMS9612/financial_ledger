나만의 가계부
=============

# 개요
***
개인용 가계부 웹사이트입니다. 본인의 아이디를 이용해 등록정보를 DB에 저장하여 소비,수입 정보를 한눈에 볼 수 있게 해줍니다.
Next14 / Tailwind / Axios / RestAPI / Express / MongoDB를 사용해서 제작한 포트폴리오 사이트입니다.
디자이너가 아니기때문에 웹사이트가 다소 깔끔하지 않은 점 양해 부탁드립니다.

# 사용 스택
***
1. Front-End
* Next.js 14
* TypeScript
* TailWind
* Recoil
2. Back-End
* Express
* TypeScript
* Node.js (NodeMon)
* MongoDB / Mongoose
