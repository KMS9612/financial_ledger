# 나만의 가계부

# 개요

---

개인용 가계부 웹사이트입니다. 본인의 아이디를 이용해 등록정보를 DB에 저장하여 소비,수입 정보를 한눈에 볼 수 있게 해줍니다.
Next14 / Tailwind / Axios / RestAPI / Express / MongoDB를 사용해서 제작한 포트폴리오 사이트입니다.

# 경로

https://financial-ledger-ten.vercel.app/
(테스트 ID를 이용해 주세요.)

1. Front-End

- Next.js 14
- TypeScript
- TailWind
- Chart.js
- Recoil
- Axios

---

2. Back-End

- Express
- TypeScript
- Node.js (NodeMon)
- MongoDB / Mongoose
